I seperated my javascript functions in to different files depending on their purpose. Rocket Launches

I used jQuery and API to collect rocket launch inforamtion from a given database. I had the user interact with the given interface search fields in order to narrow the search and return results that met all the given parameters. The user could search based on year, where the rocket was from, who the customer was, and several other fields. The JavaScript would then sort through the results, format the correct ones, and then display them in an easy to comprehend manner to the user.
